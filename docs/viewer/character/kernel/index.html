Character Definition

カーネル・レオンハルトは、ジェイド・レオンハルトの姉であり、 ZERO編における「家族の日常」「庶民生活」「守られたかった平穏」の象徴である。

しかし内部設定上の本質はそれだけではない。

彼女は同時に、日常の象徴、神性適合個体、魂回収対象、 ジェイドの倫理観の原点、喪失そのもの、後に再会する悲劇装置を一身に担う、 多層的な中核キャラクターである。

Appearance

カーネルは、ハニーゴールド〜赤橙寄りの髪を高い位置で結んだポニーテールと、 アンバーの瞳を持つ活発な少女である。

健康的で引き締まった体格をしており、生命感の強い表情と、 前へ出る意志のある視線を持つ。

第一印象は明るく快活だが、軽さではなく、 「自分が前に立つことで誰かを守ろうとする強さ」が滲む外見である。

Personality

カーネルは元気で現実的な性格をしている。 少し勝ち気で、思い立ったらすぐに行動するタイプであり、 状況判断より先に身体が動くことも多い。

ただし、その行動力は自己中心性ではなく、 他人を助ける方向へ自然に向く ことに最大の特徴がある。

面倒見が良く、他者保護傾向が強く、自分が損してでも人を助ける。 子供や弱者に自然と手を差し伸べ、市場でも教会でも頼られやすい。 誰かを守ることに理由を必要としない人物である。

Relation with Jade

ジェイドにとってカーネルは、単なる姉ではない。

彼女は、家族、日常、安心、笑い、外の世界へ出る勇気、 そして人を守る強さの原型、そのすべてを兼ねる存在である。

ジェイドが「強さとは何か」を考えるとき、 その最も原初的な答えはカーネルに由来している。

そのため彼女の死は、単なる喪失ではなく、 ジェイドにとって 託された期待の断絶 でもある。

Human Warmth / Daily Presence

後の聖骸騎士化や再会構造を成立させるため、 カーネルは生前から十分に魅力的で、人間的なキャラクターでなければならない。

教会では雑務や子供の世話を自然に引き受け、 市場では店番や荷運びを頼まれても嫌な顔をしない。

家ではジェイドをからかいながらも生活面では無意識に支え、 理屈ではなく身体感覚として「そばにいると安心する」存在である。

だからこそ、後に彼女自身が「守るために作られた兵器」へ変質する構図は、 強い悲劇性を持つ。

Soul Setting

カーネルは、本人に自覚はないが、 前世において下級天使であった魂を持つ転生体である。

このため彼女の魂には、通常の人間とは異なる 神性残滓 が宿っている。

その結果、教会のステンドグラスや祈祷空間に対する微弱反応、 魔法石ペンダントなど神性媒介物との共鳴、 神父エリオスの違和感、神性儀式痕跡への高い適応性、 テロ事件における魂回収対象としての優先性が成立する。

この時点では本人も周囲も、その本質を理解していない。

Divine Compatibility

カーネルは中央市場テロ事件の犠牲者の中でも、 極めて高い神性適合率を持つ個体である。

ただし、その適合率は単なる先天素質だけではない。

カーネルの神性適合率は、前世由来の素質だけでなく、 今世における献身性・責任感・他者保護傾向によってさらに高められていた。

つまり彼女は「元々適性があった」のではなく、 生き方そのものが魂価値を高めてしまった存在 である。

そのため彼女の魂は、単なる高適合個体ではなく、 完成されつつある器 として認識される可能性が高い。

Connection to Church Arc

教会バイト編で描かれる現象は、後の内部設定と接続する伏線である。

礼拝堂でカーネルだけが導かれるように光を受ける描写は、 単なる演出ではなく神性残滓への反応である可能性が高い。

神父エリオスは彼女を「普通の庶民の少女」として見きれておらず、 魂に残る神性に対して違和感を抱いている。

また、魔法石ペンダントが微かに反応したのは、 カーネルの魂が神性媒介物に適合したためである。

この段階で既に、外部勢力または神性観測側から 「高価値魂」として認識されていた可能性がある。

Death in the Central Market Terror

カーネルは中央市場テロ事件で死亡するが、その死は単なる巻き込まれではない。

中央市場テロにおいてカーネルが炎槍によって死亡したのは偶然ではなく、 魂損傷を抑えた状態で回収するための選別殺害である可能性が高い。

大量破壊の中で、彼女だけが炎槍によって死亡する。 炎槍は高火力攻撃ではなく、魂分離・選別・回収に適した神性攻撃である可能性がある。

カーネルの死は偶然の悲劇ではなく、 回収を前提とした選別悲劇 として設計されるべきである。

Last Words / Residual Voice

カーネルの最期には、ジェイドへ向けた呼びかけが残る可能性が高い。

「ジェイド……」「逃げて……」「大丈夫だから……」「強くなって……」

こうした最後の言葉や意志は、死後の人格加工を経ても完全には消えず、 深層魂の残響として残る核になる。

Soul Recovery

カーネルの魂は通常の死者と同じようには処理されない。

回収工程は、炎槍による致命傷、魂の分離、神性媒介による保全、 外部機関または黒幕国家による回収、そして記憶封鎖・人格加工工程への移行である。

肉体は死ぬが、魂は終わらない。 彼女は「死者」ではなく、 利用可能な神性資源 として回収される。

Connection to Sanginaria

カーネルの魂回収は、サンギナリア王国の黒幕構造と強く結びつく。

サンギナリアは表向きには高貴で伝統的な王国であるが、 内部では神性技術・魂加工・人格上書き制度を保持している。

そのためカーネルのような高適合魂は、犠牲者ではなく、 再利用可能な 国家資源 として扱われる。

中央市場テロ事件は、 宗教テロ・国家工作・魂資源回収作戦の三層が重なった事件として再定義される。

Sacred Remains Knight Conversion

カーネルは死後、神性技術によって聖骸騎士へ再構成される候補となる。

推奨分類は、聖骸騎士、殉天兵、昇華体。

魂は保持され、肉体は再構成または神性媒介体に置換される。 記憶は消去ではなく暗号化・封鎖され、命令系統は神性儀式・王国中枢へ直結する。

カーネルの魂の本質は破壊ではなく保護に適している。 だからこそ、その魂を対英雄兵器に変えることは、 彼女の本質そのものを踏みにじる行為である。

Residual Personality Core

聖骸騎士化後も、カーネルの人格が完全に消えたわけではない。

“ジェイド”にのみ反応する深層の残響として残留する可能性がある。

それは、一瞬の躊躇、涙、呼吸や鼓動の乱れ、攻撃の鈍り、 無意識の庇い動作、命令系統の微細な乱れとして現れる。

この残響があることで、再会は単なる敵味方の衝突ではなく、 消されきらなかった家族愛との対面 になる。

Combat Memory Residue

聖骸騎士となった後も、戦闘動作の一部には生前の身体感覚が残っている可能性がある。

左手の使い方、重心移動、咄嗟に庇う動き、足運び、 抱き寄せる時と同じ腕の軌道、構えに入る前のわずかな癖。

他者にはただの戦技に見えるが、 ジェイドだけはそれを「姉の癖」として見抜ける。

これは再会シーンにおいて極めて強い演出要素となる。

Meaning for Jade

ジェイドにとってカーネルは、単なる守るべき姉ではない。

彼女は、良心の原型、強さの原型、家族の象徴、喪失の核、 真相追求の理由、再会すべき過去、そのすべてを兼ねている。

ジェイドにとってカーネルは、 「強さとは誰かを支えるために使うものだ」という価値観の原型である。

その原型が敵として現れることは、 ジェイドの人格そのものへの攻撃となる。
