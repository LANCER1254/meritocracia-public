01

概要

セレス・オブシディアンは、中央警備隊 ZG に所属する副官であり、 首席監察補佐として制度運用の実働を担う存在である。

国家試験不正の摘発、士官学校監査、貴族腐敗調査など、 国家制度の現場執行に関わる任務を担当し、 団長ルチアーノの判断を実務として落とし込む側近ポジションにある。

03

キャラクターコンセプト

ルチアーノが制度の骨なら、セレスは制度の刃である。

セレスは単なる机上官僚ではない。 記録を精査し、現場へ出て、不正を切り、感情より規則を優先する “制度執行の実働側”として設計される。

その厳しさは冷酷さと隣接して見えるが、 本質的には制度を守ることへの本気さから来るものであり、 無機質ではなく職務への熱量を内包している。

05

関係性

ユミナとの関係

同世代に近いが、進んだルートが違う相手。 セレスはユミナの実力を認めつつも、感情が混じる点を危ういと見る。 一方でユミナは、セレスの正しさを理解しながらも、 制度を信じすぎていて冷たいと感じている。

ルチアーノとの関係

ルチアーノ直属の副官として、 最終判断者の意志を実務として執行する側近。 信頼はあるが甘さはなく、 厳格な上司と有能な副官という構図を基本とする。
