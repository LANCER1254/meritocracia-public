ルチアーノ・コバルト （仮）

ZG団長 / 管理責任者 / 制度運用型指揮官

ZG COMMANDER SYSTEM CONTROL FINAL DECISION
01

概要

ルチアーノは、中央警備隊 ZG を率いる制度運用型の責任者である。 前線で武名を轟かせる英雄ではなく、 国家試験運営、士官学校監査、貴族腐敗調査、内務・封鎖判断などを司る、 ZGの事務・監察・統制の頂点に立つ人物として設計される。

感情ではなく規則を重視するが、単なる冷酷な官僚ではない。 秩序維持を本気で信じ、その責務を引き受ける側の人間である。

03

キャラクターコンセプト

感情ではなく制度で国家を支える、中央警備隊の頑固な管理責任者。

ルチアーノは「制度の骨」として設計される。 彼自身は剣よりも書類、激情よりも規則、英雄譚よりも秩序の持続を優先する。

その在り方はジェイド視点では恐ろしく映るが、 無能さや惰性ではなく、国家を支える責任の重さから来るものとして描かれる。

06

他キャラとの差別化

ロータスとの対比

ロータスより冷たい。 同じ国家上層でも、包容ではなく管理・判断・統制の側に重心がある。

ガーネットとの対比

ガーネットより事務的。 情熱や現場圧ではなく、制度そのものを回す責任者として差別化される。

セレスとの関係

セレスより上位の最終判断者。 セレスが制度の刃なら、ルチアーノはその刃を振るう根拠と責任を担う。

ジェイド視点

「怖いが無能ではない大人」。 子ども視点では息苦しいが、単純な悪役には見えない存在として機能する。
