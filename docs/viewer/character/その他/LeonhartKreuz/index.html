レオンハルト・クロイツ

Leonhart Kreuz

守るために迷いを切り捨て、最短で局面を終わらせる。 彼は英雄ではない。だが、炎の後に残る混乱と責任を、 誰より早く処理する「制度の執行者」である。

  • 強さとは、切る責任を持つことだと定義する

POSITION IN STORY

レオンハルト・クロイツは、「制度の冷たさ」を単なる設定ではなく、 人間の顔として提示するためのキャラクターである。 彼は悪役ではない。だが救いの顔もしていない。 国家運用の現実を、最も鋭利な形で少年世代に突きつける。
