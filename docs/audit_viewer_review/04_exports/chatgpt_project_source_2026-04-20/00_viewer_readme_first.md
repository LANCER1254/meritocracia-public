# ChatGPT Project Source Set
作成日: 2026-04-20

## 目的
このフォルダは、ChatGPTのプロジェクト情報源へ投入するために整理した内部用資料セットである。

## 方針
- viewer全体をそのまま渡すのではなく、公開理解と全体把握に必要な資料だけを抽出
- ネタバレや内部作業ログは極力除外
- 公開導線、作品導線、地図、DB入口、共有導線を中心に整理

## ファイル一覧
- 01_public_hub.txt
  - official-site 由来の公開ハブ資料
- 02_character_public.txt
  - 公開キャラ関連の資料束
- 03_story_public.txt
  - story系公開導線資料
- 04_map_public.txt
  - map系公開導線資料
- 05_shared_public.txt
  - 共有用公開資料の中核
- 06_db_public_core.txt
  - dbの入口ページ中心に絞った公開理解用資料
- 07_viewer_classification.md
  - viewerを公開候補 / 内部候補 / 除外候補で分類したメモ
- 08_docs_summary.txt
  - docs全体とviewer配下の件数サマリ

## 使い方
- ChatGPT情報源へ投入する一次セットとして使用
- 後から25ファイル以内に圧縮する際の基準セットにも使う
- 募集用viewや公開導線設計の前提理解に使う

## 注意
- これは公開ページそのものではなく、公開ページを安全に設計するための内部整理資料である
- production / memo / archive などの内部性が強い層は原則含めていない
