# ChatGPT Final Source Set
作成日: 2026-04-20

## 目的
このフォルダは、ChatGPT のプロジェクト情報源へ投入するために整理した最終資料セットである。  
viewer 側の公開導線資料と、Meritocracia_Data_Hub 側の世界観・物語コア資料を統合し、
作品全体像・公開構造・制度設定・物語進行を横断的に理解できるようにすることを目的とする。

## 収録方針
- viewer 全体や Data Hub 全体を丸ごと渡すのではなく、理解に必要な中核資料だけを抽出
- production / memo / archive / backup / old など内部性の強い層は原則除外
- 公開導線、ストーリー導線、地図導線、DB入口、世界観制度コア、Reboot本編コアを優先
- 将来的に 25 ファイル以内へ再圧縮する際の基準セットとしても使う

## ファイル一覧

### Viewer 系
- 01_public_hub.txt
  - official-site 由来の公開ハブ資料
- 02_character_public.txt
  - 公開キャラ関連の資料束
- 03_story_public.txt
  - story 系公開導線資料
- 04_map_public.txt
  - map 系公開導線資料
- 05_shared_public.txt
  - 共有用公開資料の中核
- 06_db_public_core.txt
  - db の入口ページ中心に絞った公開理解用資料
- 07_viewer_classification.md
  - viewer を公開候補 / 内部候補 / 除外候補で分類したメモ
- 08_docs_summary.txt
  - docs 全体と viewer 配下の件数サマリ

### Data Hub 系
- 09_datahub_world_system_core.txt
  - 世界観・制度・神話・年表・試験制度の中核資料
- 10_datahub_reboot_story_core.txt
  - Reboot 本編・完成原稿・プロローグ系の中核資料
- 11_datahub_reboot_plot_core.txt
  - Reboot の未来進行・章構造・プロット中核資料

## 使い方
- ChatGPT プロジェクト情報源への一次投入セットとして使用する
- 募集用 view / 公開導線 / ゲーム企画ページ設計の前提理解に使う
- Claude / アンチグラビティへ渡す資料を選別する際の母集団として使う
- 後から追加投入・再圧縮する場合の基準セットとして使う

## 注意
- これは公開ページそのものではなく、公開ページを安全に設計するための内部整理資料である
- production / memo / archive / old / backup などの内部作業層は原則含めていない
- 一部の資料は公開導線理解のために HTML / CSS / JS を含む
- より深い内部設定が必要な場合は、Data Hub 本体や production 系資料を別途監査して追加する
